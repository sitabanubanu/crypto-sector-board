# 01 - 产品需求

## 1. 用户画像
- 加密市场短线交易者
- 编程零基础，依赖 AI 协作开发
- 主要诉求：每天开盘前快速判断哪些板块走强，决定当日交易标的

## 2. 核心使用场景
1. **每日复盘**：UTC 0 点之后打开网页，一眼看到上一日各板块强弱排序
2. **板块对比**：直观判断 DeFi、AI、Memecoin 等板块今天谁更强
3. **币种细节**：鼠标悬停具体币种查看其高低、振幅、涨跌

## 3. 功能列表（MVP）
- [ ] 自定义板块清单（手工维护 `data/sectors.json`）
- [ ] 每日 UTC 00:30 自动抓取上一日 OHLC + 市值（GitHub Actions）
- [ ] 计算单币指标：最高、最低、振幅、收益率、波动率
- [ ] 计算板块指标：按市值加权平均
- [ ] Treemap 嵌套渲染（板块外层 + 单币内层）
- [ ] 颜色编码涨跌（绿涨红跌、深浅表幅度）
- [ ] 鼠标悬停 Tooltip 显示单币详情
- [ ] 顶部 Header 显示数据日期 + 最后更新时间

## 4. 非功能需求
| 维度 | 目标 |
|---|---|
| 性能 | 首屏 < 2s，Treemap 交互流畅 |
| 可用性 | Vercel 99% 可用即可（个人工具） |
| 可维护性 | 板块清单纯数据维护，不改代码 |
| 可观测性 | GitHub Actions 失败邮件通知 |
| 兼容性 | 桌面 Chrome/Edge/Safari + 手机浏览器 |

## 5. MVP 范围（已确认）
- ✅ "昨日完整快照" 主屏
- ✅ 板块清单自定义
- ✅ UTC 0 点切日
- ✅ Vercel 公开网址 + GitHub Actions 自动化
- ❌ 3天/周/月历史对比页（数据归档但不渲染）
- ❌ 排序表格 / K 线下钻 / 网页编辑板块清单
