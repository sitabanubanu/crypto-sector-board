# 04 - 视觉规范

## 1. 整体风格
**浅色背景 + 红涨绿跌**（贴合国内主流交易软件，如同花顺/东方财富）。参考用户提供的 mock 设计图。

## 2. 调色板

| 用途 | 颜色 | Hex |
|---|---|---|
| 页面背景 | 极浅灰白 | `#f5f6f8` |
| 卡片/Header 背景 | 纯白 | `#ffffff` |
| 主文字 | 近黑 | `#1f2328` |
| 次要文字 | 中灰 | `#6b7280` |
| 边框/分隔线 | 浅灰 | `#e5e7eb` |

### 涨跌色阶（红涨绿跌）

| 涨跌幅区间 | 颜色名 | Hex |
|---|---|---|
| ≥ +8% | 深红 | `#c81e1e` |
| +4% ~ +8% | 红 | `#e53e3e` |
| +1% ~ +4% | 中红 | `#fc8181` |
| 0 ~ +1% | 浅红 | `#fed7d7` |
| 平盘 0% | 中性灰 | `#e5e7eb` |
| -1% ~ 0 | 浅绿 | `#c6f6d5` |
| -4% ~ -1% | 中绿 | `#68d391` |
| -8% ~ -4% | 绿 | `#38a169` |
| ≤ -8% | 深绿 | `#22543d` |

### 文字色（在彩色方块上）
- 红色块上：白色文字 `#ffffff`
- 绿色块上：白色文字 `#ffffff`
- 浅红/浅绿/平盘块上：深色文字 `#1f2328`

## 3. 涨跌色映射函数
```ts
function returnPctToColor(pct: number): string {
  if (pct >= 0.08)  return "#c81e1e";
  if (pct >= 0.04)  return "#e53e3e";
  if (pct >= 0.01)  return "#fc8181";
  if (pct >= 0)     return "#fed7d7";
  if (pct >= -0.01) return "#c6f6d5";
  if (pct >= -0.04) return "#68d391";
  if (pct >= -0.08) return "#38a169";
  return "#22543d";
}

function textColorOnBlock(pct: number): string {
  const absPct = Math.abs(pct);
  return absPct >= 0.01 ? "#ffffff" : "#1f2328";
}
```

## 4. Treemap 布局

### 视图模式
- **详细模式（默认）**：板块外框 + 内部单币子方块。子方块按市值面积排布。
- **总览模式**：只显示板块外框、不显示子方块。整个板块涂一个加权涨跌色。
- 顶部 Header 提供 toggle 切换。

### 板块外框（详细模式）
```
┌────────────────────┐
│ Layer 1   +1.23%   │  ← 顶栏（板块名 + 加权涨幅）24-32px 高
├────────────────────┤
│ ┌────┐ ┌────┐      │
│ │ETH │ │SOL │      │  ← 内部子方块（单币）
│ │+2.1│ │+3.4│      │
│ └────┘ └────┘      │
└────────────────────┘
```

### 子方块（单币）
```
┌──────────┐
│   BTC    │  ← 币 symbol，加粗，方块大时显示
│  +1.46%  │  ← 涨跌幅
└──────────┘
```

## 5. 面积映射
- **板块面积**：`Math.sqrt(板块总市值)` —— 压头部权重，BTC 不会占半屏
- **单币面积**（板块内部）：`Math.sqrt(单币市值)` —— 同理

## 6. 字号/间距

| 元素 | 字号 | 字重 |
|---|---|---|
| 板块名（顶栏） | 14-20 px 自适应面积 | 600 |
| 板块涨幅（顶栏） | 14-18 px | 700 (tabular-nums) |
| 单币 symbol | 11-16 px 自适应 | 700 |
| 单币涨幅 | 10-14 px | 500 |
| Header 主标题 | 22 px | 700 |
| Header 副标题 | 13 px | 400 |

间距：
- 板块间留白：6 px
- 单币间留白：2 px
- 方块内 padding：4-8 px

## 7. Tooltip（鼠标悬停）
显示在悬停币上方，包含：
- 全名 + Symbol
- 开盘 / 最高 / 最低 / 收盘
- 24h 涨跌 %
- 振幅 %
- 流通市值（亿/B）

样式：白底、浅灰边框、阴影、12 px 字、左对齐。

## 8. 响应式

| 断点 | 宽度 | 行为 |
|---|---|---|
| Desktop | ≥ 1280 px | 完整 treemap，主视图占满全屏 |
| Tablet | 768 - 1280 px | 字号/间距略缩小 |
| Mobile | < 768 px | 缩成单列纵向板块堆叠，子方块自动隐藏，强制总览模式 |

**桌面优先**：本项目主战场是 PC 复盘。

## 9. 字体
- 中文：系统默认（PingFang SC / Microsoft YaHei）
- 数字：`font-variant-numeric: tabular-nums`（等宽，对齐）

## 10. 可访问性
- 关键涨跌幅在 Tooltip 里以文字"涨/跌 X%"重复，不只靠颜色传达信息
- 切换按钮带 aria-label
